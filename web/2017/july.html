July 2017 Hacker News Hiring Trends

React wins second in a row; CoffeScript drops out

Show: or compare

July Highlights

Saturday, July 22, 2017

For the second month in a row, React is the top technology in the Hacker News Who is Hiring thread. While this demonstrates how strong React's hold on the industry is right now, how long can a JavaScript framework remain on top though? History has not been kind to JS frameworks, so my guess is that React will start dropping down by the end of year. With Vue.js coming in as the top riser, it's clear the competition is coming.

The JavaScript ecosystem has always moved fast, which has provided many compelling story lines for this data over time. But, that fast pace has brought not only breakout hits, but also means some are left behind. We see that this month with both CoffeScript and ClojureScript dropping to zero. With TypeScript gaining momentum, so languages that compile to JS are not losing their appeal completely, but the industry does appear to be moving on somewhat into new territory with ES6.

The rest of the top 10 remains pretty similar to the last few months with some slight shuffling. Containers are hot however with Docker fast approaching the Top 10 and Kubernetes skyrocketing close to the top 20.
