October Highlights

Friday, October 3, 2014

JavaScript, JavaScript, JavaScript. For the sixth month in 2014, JavaScript takes the top spot. Python is still hanging around in its usual #2 spot. In terms of big movers this month, Rails picked up a big gain to take #3 which, I believe is the highest its ever risen. Also, Java made a big jump to #4 after falling out of the top 10 last month.

Programming Languages

Compare JavaScript, Python, Ruby and Java
Compare C++, Scala, Clojure and Go

Of note with programming languages, Scala tied up with C++ as the fifth most popular language. Combine it with Java (#3 programming lang) and Clojure (#8 programming lang) and the JVM ecosystem is showing some strength.

Server-side Frameworks

Compare Rails, node.js, PHP and Django

It looked like node.js had all the momentum in August, beating Rails for the first time. But, for the past two months, Rails has roared back to take a commanding lead, so looks like the August win for node.js as an anomaly.

JavaScript Frameworks

Compare AngularJS, Backbone, Ember and jQuery

The slide of jQuery continues with both AngularJS and Backbone.js now more popular than it.
