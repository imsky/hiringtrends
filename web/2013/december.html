About

This chart represents trends in languagues, frameworks and technologies used to develop modern software applications as indicated by Hacker News activity. Using the HN Search API and a dictionary of software terms, all of the the comments (job postings) in the "whoishiring" submissions from April 2011 to December 2013 were processed to count the number of times each software term was mentioned.
