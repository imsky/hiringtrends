Python wraps up a strong year, React reaches top 5

Sunday, December 27, 2015

Python capped off an impressive year, winning for the 9th time in 2015. This is actually really interesting since it's not used for mobile apps, or client-side on web apps, which are the 2 other dominant trends of the year. Server-side developers and data scientists can take heart that much work remains out there.

Speaking of client-side, the quick-rising React joined the top 5 this month, notable for the fact that it just started showing up in postings early last year, while the other top technologies have been around since the hiring threads started in 2011.
