Android Rises to the Top

Saturday, November 7, 2015

We have a new champ! It was close, with only 1 or 2 mentions separating the top 4 but Android is #1, highlighting mobile app development's ascension. Throughout the history of these rankings it's consistently been a battle between the multi-use langauages JavaScript and Python for the top spot. Now, with Android #1 and iOS #3, mobile is joining the battle too.

In other big news and speaking of climbs to the top, React completed one too, surpassing Angular in the front-end javascript framework cateogory. Based on its trajectory, it was easy to see this coming, but we can now mark it down. Late 2015 saw React as the most popular front-end framework.
